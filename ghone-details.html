Ghone

Project information

  • Category: iOS Native App
  • Developer Team: Croissant
  • Project Timeline: Jun 2023 - Aug 2023

App Description

An app that helps adults who are alone at night and scared of the sound associated with the presence of ghosts feel reassured with the presence of a virtual companion as a support by providing several methods such as comforting words, breathing techniques, relaxing audio, and riddles.

Problems

Did you know that ghost stories and movies can make people feel paranoid when encountering certain triggers? These triggers could be anything, such as hearing weird sounds or seeing random shadows. These stories and movies fuel our imagination and make us more sensitive to our surroundings, causing us to hear certain things at night and cannot help but associate those sounds with the existence of ghosts. When in fear, we often don't know what to do, and it can lead to difficulty sleeping, reducing productivity the next day. One way to reduce fear is to find a distraction, and this distraction can take the form of support from someone. However, as adults, we become more aware that it's not polite to disturb others late at night since most cases of particular fear happen late at night. What if you have a friend who can provide support 24/7 and can distract you from your fear?

Process

My Role is as a coder

In my role as the lead coder, I guided a team of developers through the project, focusing on effective collaboration and teamwork. I used SwiftUI to build the app's logic and views, organizing them with Clean Architecture and MVVM patterns for simplicity in maintenance and modularity.

For improved chat functionality, I integrated the ChatGPT API and Text-to-Speech feature, aiming for a user-friendly experience through the Ghone feature. Additionally, I collaborated closely with the team throughout the development lifecycle of new features, from ideation and research to implementation, fostering transparent communication and effective teamwork.

Tools and Technology

SwiftUI

SwiftUI is used to create the user interfaces. I chose SwiftUI because most of my teammates more familiar and comfortable using SwiftUI.

AVFoundation

I utilized AVFoundation for recording and playing audio, as well as implementing the Speech-to-Text feature to enable users to interact with Ghone.

CoreAnimation

CoreAnimation is employed to create captivating character animations, enriching the user experience.

OpenAI API & ElevenLabs API

OpenAI API digunakan untuk generate text yang digunakan sebagai jawaban dari interaksi user kepada ghone melalui fitur chat. ElevenLabs API digunakan untuk mengubah text menjadi audio yang digunakan sebagai jawaban dari ghone.

Github

GitHub is used for version control and collaborate with my teammates.

Versions

The application underwent multiple iterations throughout its development process, involving continuous refinement through bug fixes and feature optimizations. These changes were instrumental in enhancing the overall performance and user experience.

v0.1.0

In this version, I implemented the Minimum Viable Product (MVP) of the application, featuring functionalities such as chatting with Ghone, the 4-7-8 breathing technique, relaxing audio, and riddles. I leveraged OpenAI's API for the chat with Ghone feature and ElevenLabs' API for the text-to-speech capability. To ensure a robust development process, I utilized the MVVM design pattern as a solid foundation. However, it's worth noting that this version still requires bug fixes and code optimizations for further refinement.

v0.2.0

In this version, I undertook code refactoring, merging Clean Architecture with MVVM to ensure modular and easily maintainable code. Additionally, this release includes bug fixes and animation optimizations, enhancing the overall stability and performance of the application.

The Challenges

Implementing MapKit and CoreLocation into our application posed a significant challenge for both myself and the team. Prior to this project, none of us had hands-on experience in integrating mapping services, making it a learning curve that required dedicated effort and collaboration. Overcoming this obstacle marked a valuable milestone in our journey.
