Project information

  • Category: iOS Native App
  • Project Timeline: Jun 2023 - Aug 2023

App Description

Here is a web based application called HafizOnline that i built with team as a web project in the web programming class. This website is made for people who want to memorize the Quran since the feature are Read Quran, Memorize and Repeat, Event and Article.

Problems

Many Muslims, eager to independently memorize the Quran, face a significant challenge in assessing the fluency and accuracy of their recitation without external guidance. A considerable number of these individuals lack access to a teacher or mentor who can listen and provide feedback on their memorization efforts. To address this predicament, the HafizQuran website has been developed, aiming to facilitate Muslims in their independent Quranic memorization journey. The platform offers a range of features, including tools for memorization assistance, the submission of memorized verses, proficiency assessments, and a wealth of informative articles and events. By providing a supportive online environment, HafizQuran strives to enhance the memorization experience for those who may not have access to traditional teaching methods, serving as a valuable resource on their memorization journey.

Process

My Role is as a coder

In my dual role as a Full Stack Developer and UI/UX Designer, the project's process unfolded with a series of responsibilities. To begin, I designed the user interface meticulously using Figma, ensuring a visually appealing and user-friendly design. The next step involved translating this design into functional code using HTML, CSS, and JavaScript, blending aesthetics with seamless functionality. Simultaneously, I took charge of crafting the project's database structure, designing it with efficiency and scalability in mind using MySQL. This step laid the groundwork for robust data management throughout the application. In the backend development phase, I seamlessly integrated the user interface with the database, employing PHP and MySQL for a cohesive and responsive system. This collaborative effort between design and development elements aimed to create an engaging and efficient user experience without compromising on the project's technical integrity.

Tools and Technology

HTML, CSS & JavaScript

HTML for structuring the content. CSS for styling and layout design. JavaScript for dynamic and interactive elements on the frontend. Together, these technologies shape the user interface, ensuring a visually appealing and responsive website.

PHP

PHP is utilized for backend development, specifically in integrating and processing data to and from the MySQL database. It plays a pivotal role in ensuring smooth communication between the frontend and the database, enhancing the overall functionality of the website.

MySQL

MySQL serves as the primary database management tool. It is instrumental in designing and structuring the database, providing a robust foundation for storing and retrieving data efficiently. MySQL plays a crucial role in managing the backend data operations seamlessly.

Github

GitHub is used for version control and collaborate with my teammates.
